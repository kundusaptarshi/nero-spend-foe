Introduction

Welcome to NeroSpendFoe. Unlike static budget apps, Nero is dynamic. Every transaction instantly updates your forecast and daily limits for the rest of the month, ensuring you never run out of funds.

Why Nero?

Automatic Calibration

Spend heavy today? Nero lowers future limits automatically. Save today? Your future limits go up.

100% Safe & Private

No banking details required. No login. No server tracking. Your data is yours alone.

PWA Architecture

Offline First

Works without internet.

Zero Install

No app store downloads needed.

Setup & Install

Installation Guide

Android (Chrome)

Tap the menu (⋮) and select "Install app" or "Add to Home screen".

iOS (Safari)

Tap the "Share" icon (bottom center) and select "Add to Home Screen".

Quick Start

1. Initialize

Tap 'Start New Month' to set your total budget.

2. Configure

Set Currency & Name in Settings.

3. Track

Log expenses daily. The system auto-adjusts limits.

Core Operations

Transaction Entry

Logging

Enter amount and Confirm. This deducts from your Limit Today.

Shortcuts

Check 'Save Shortcut' to create quick buttons for frequent expenses.

Dashboard Analytics

Limit Today

Your live spendable balance. Changes dynamically based on spending.

Original Limit

Your planned baseline for the day.

Monthly Limit Remaining

Total funds left. Hitting zero blocks spending.

Advanced Controls

Monthly Limit Forecast

View future limits. Edit the Original Plan (Orange) column to adjust specific days; the system redistributes the difference across the rest of the month.

Top-Up Monthly Limit

Inject unexpected income into your pool. Only available when a month is active.

Start New Month

Resets data for a new cycle. Offers to carry forward remaining funds if available.

History Logs

Track your income sources (Fund History) and daily expenditure logs (Spend History).

Configuration

Settings

Currency

Set your preferred currency symbol.

A hard reserve to ensure you never run completely dry.

Troubleshooting

Common Errors

"Start a new month first"

Buttons are locked because the month has ended or hasn't started. Tap 'Start New Month' to unlock.

"Safety Halt"

Your transaction is too large. It violates the Minimum Daily Limit for future days.

"Limit is 0.00"

You have exhausted your daily allowance.

"It will not leave any limit..."

Manual forecast edit rejected because it drains the pool below safety levels.

"Limit too low"

Initialization rejected; starting pool cannot cover the Minimum Daily Limit for the month.

"Error: Amount needs to be numbers"

Please enter a valid numeric value.
